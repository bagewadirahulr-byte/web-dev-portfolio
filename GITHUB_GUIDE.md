# 🔥 Web Dev Portfolio — GitHub Upload Guide

## Step 1: Open Terminal in VS Code
Press **Ctrl + `** (backtick) to open the terminal inside VS Code.

---

## Step 2: Go to your project folder
```bash
cd /Users/rahulrbagewadi/.gemini/antigravity/scratch/web-dev-portfolio
```

---

## Step 3: Initialize Git
```bash
git init
```

---

## Step 4: Add all files
```bash
git add .
```

---

## Step 5: Make your first commit
```bash
git commit -m "🔥 Initial commit - Demon Slayer Portfolio + 3 Web Projects"
```

---

## Step 6: Create a new repo on GitHub
1. Go to [github.com](https://github.com) → Click **"New"**
2. Name it: `web-dev-portfolio`
3. Keep it **Public**
4. **Do NOT** add README (we already have files)
5. Click **Create Repository**

---

## Step 7: Connect and Push
Copy the commands from GitHub and run them. They look like this:
```bash
git remote add origin https://github.com/YOUR_USERNAME/web-dev-portfolio.git
git branch -M main
git push -u origin main
```

---

## ✅ Done! Your portfolio is live on GitHub!

To also deploy it live (free):
- Go to your GitHub repo → **Settings** → **Pages**
- Source: **main branch** → folder: **/portfolio**
- Your site will be live at: `https://YOUR_USERNAME.github.io/web-dev-portfolio/`
